# Pokédex Tutorial

A very simple pokedex web application example.

Uses the following APIs and technologies:

- https://pokeapi.co/docs/v2.html
- Node `v10.x`
- React `v16.x`
  - with Create React App `v2.1.x`
- SASS for css-styling


## Getting Started

```bash
$ git clone https://github.com/lemonpole/pokedex-tutorial

$ cd pokedex-tutorial
$ yarn && yarn
```
